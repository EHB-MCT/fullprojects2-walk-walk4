# Walk Brussels

Walk Brussels is een eenvoudige mobiele webapp om een obstakel op de stoep te melden. De gebruiker vult eerst contactgegevens in, duidt daarna een locatie aan, kiest een categorie, voegt eventueel een foto toe en krijgt op het einde een bevestiging.

## Wat zit er in de site?

- Een formulier voor naam, e-mail en telefoonnummer.
- Een interactieve kaart van Brussel.
- Een GPS-knop die alleen werkt als de gebruiker zelf op de knop klikt.
- Een adresveld als alternatief voor GPS.
- Een sleepbare rode speld om de locatie van de melding aan te duiden.
- Een blauw bolletje om de GPS-locatie van de gebruiker te tonen.
- Een categoriepagina voor het type obstakel.
- Een fotostap om bewijs toe te voegen.
- Een bedankpagina met een korte samenvatting.

## Bestanden

- `index.html`: de structuur van de pagina's.
- `syle.css`: de styling en huisstijl.
- `script.js`: de interactie, kaart, formuliercontrole en navigatie.
- `afbeeldingen/`: logo en illustraties.

## Bronnen

- Kaart: OpenStreetMap tiles.
- Kaartbibliotheek: Leaflet 1.9.4 via unpkg.
- Lettertype: Nunito Sans via Google Fonts.
- Afbeeldingen: lokale bestanden in de map `afbeeldingen`.
- gebruik gemaakt van chatgpt voor code kaart en gps locatie, debuggen en code herschrijven.
